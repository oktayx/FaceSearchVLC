FaceSearchVLC

Real-time face recognition while watching video on VLC. Identify actors on screen with instant toast notifications and one-click IMDB lookup.

View on GitHub Quick Start

How It Works

Four simple steps from VLC snapshot to face identification.

Folder Watcher

Monitors VLC's snapshot directory for new image files using FileSystemWatcher.

Face Recognition

Each captured screenshot is uploaded to a local face recognition API service.

Toast Notification

A Windows 10 toast notification displays the recognized person's name, category, and photo.

IMDB Lookup

Clicking the notification launches Chrome and searches IMDB for the recognized person.

Features

Everything you need for real-time face identification while streaming.

⚡

Real-time Detection

Automatically detects new VLC snapshots and processes them instantly with no manual intervention.

🔔

Windows Toast Notifications

Native Windows 10 notifications show the person's name, category, and captured face image.

🎬

IMDB Integration

One-click notification action opens an IMDB search for the recognized actor or person.

⚙

Configurable Threshold

Adjust the recognition sensitivity via the API threshold parameter to control match strictness.

📁

Automatic VLC Config

Reads the VLC snapshot folder path directly from the VLC configuration file.

🚀

Lightweight

Built on .NET Framework 4.5.2 with minimal dependencies, runs quietly in the background.

Prerequisites

Make sure you have the following before getting started.

  • Windows 10+
  • .NET Framework 4.5.2
  • Visual Studio 2019+
  • VLC Media Player
  • Face Recognition API (port 5555)

Quick Start

Get up and running in three steps.

# 1. Clone the repository
git clone https://github.com/oktay/FaceSearchVLC.git
cd FaceSearchVLC

# 2. Build the solution
msbuild nfwFolderCheckGithub.sln /p:Configuration=Release

# 3. Run the app
# Start VLC, play a video, press Shift+S to snapshot
